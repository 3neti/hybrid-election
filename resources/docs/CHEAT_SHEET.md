# Cheat Sheet

Ballots from seed
Signatures from a directory
```bash
php artisan migrate:fresh --seed
php artisan prepare-er
php artisan certify-er --dir=storage/app/public/signatures
php artisan er:print-pdf --paper=legal --payload=minimal  
php artisan close-er
```

Ballots from command line using xargs
Signatures from a command line
```bash
php artisan migrate:fresh
php artisan preflight-er --force
xargs -L1 -I{} -S 65536 php artisan app:cast-ballot --local "{}" < resources/docs/ballots.txt
php artisan prepare-er
php artisan certify-er "uuid-juan|ABC123" "id=uuid-pedro,signature=ABC123" "id=uuid-maria,signature=ABC123"
php artisan er:print-pdf --paper=legal --payload=minimal  
php artisan close-er
```

Ballots from command line using while
Signatures from a command line
```bash
php artisan migrate:fresh
php artisan preflight-er --force
while IFS= read -r line || [ -n "$line" ]; do
  php artisan app:cast-ballot --local "$line"
done < resources/docs/ballots.txt
php artisan prepare-er
php artisan certify-er "uuid-juan|ABC123" "id=uuid-pedro,signature=ABC123" "id=uuid-maria,signature=ABC123"
php artisan er:print-pdf --paper=legal --payload=minimal  
php artisan close-er
```
